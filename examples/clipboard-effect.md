# Clipboard Effect

Use a clipboard effect when the backend needs to compute the text that should be
copied.

## Files Used

- `src/emdash/preview-url-actions.ts`: define the provider plugin, manifest
  route, and backend action route.
- `astro.config.mjs`: register `actionsPlugin()` and `previewUrlActions()` in
  the EmDash plugin list. Add `preview-url-actions` to
  `actionsPlugin({ providers })` if you want the dashboard to discover it too.
- Target collection schema file, for example `seed/schema.ts`,
  `seed/site.seed.json`, or the project-local script that defines EmDash
  collections. Add the field object to that collection's `fields` array.

## Backend Code

Create this provider module at `src/emdash/preview-url-actions.ts`:

```ts
// src/emdash/preview-url-actions.ts
import { definePlugin, type RouteContext } from "emdash";
import { defineActionsManifest, type ActionsManifest } from "@bnomei/emdash-actions";

export function previewUrlActions() {
  return definePlugin({
    id: "preview-url-actions",
    version: "0.1.0",
    routes: {
      actions: {
        handler: previewUrlActionsManifestRoute,
      },
      "entry/preview-url": {
        handler: previewUrlRoute,
      },
    },
  });
}

async function previewUrlActionsManifestRoute(): Promise<ActionsManifest> {
  return defineActionsManifest({
    actions: [
      {
        id: "entry.copyPreviewUrl",
        label: "Copy preview URL",
        route: "entry/preview-url",
        method: "POST",
        placement: "field",
        icon: "clipboard",
        tone: "info",
        contextKey: "entryId",
        contextValueKey: "entryId",
      },
    ],
  });
}

async function previewUrlRoute(ctx: RouteContext) {
  const payload = (await ctx.request.json().catch(() => ({}))) as {
    entryId?: string;
  };
  const serverTime = new Date().toISOString();
  const previewUrl = ctx.url(`/preview/${encodeURIComponent(payload.entryId ?? "new")}`);

  ctx.log.info("Created preview URL", {
    previewUrl,
    serverTime,
    userAgent: ctx.requestMeta.userAgent,
  });

  return {
    ok: true,
    status: 200,
    severity: "success",
    message: `Preview URL generated on the server at ${serverTime}.`,
    effects: {
      clipboard: {
        text: previewUrl,
      },
    },
    serverTime,
  };
}
```

## Field JSON

Put this object inside the target collection schema's `fields` array, for
example in a seed JSON file or a project-local schema setup script.

```json
{
  "slug": "preview_action",
  "label": "Preview Action",
  "type": "string",
  "widget": "actions:button",
  "options": {
    "mode": "run",
    "provider": "preview-url-actions",
    "manifestRoute": "actions",
    "action": "entry.copyPreviewUrl",
    "label": "Copy preview URL"
  }
}
```

## Frontend Behavior

The button calls the provider route, receives `effects.clipboard.text`, writes
that text to the browser clipboard, and shows the returned message as temporary
button feedback.

The copied text is generated by backend TypeScript. The route uses `ctx.url()`
to build a site-aware absolute preview URL and can inspect request metadata from
`ctx.requestMeta`.
